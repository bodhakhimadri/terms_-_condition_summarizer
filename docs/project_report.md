# Terms & Conditions Summarizer

## Problem Statement
Legal Terms & Conditions are lengthy and difficult for users to understand.
Most users agree without reading, which may lead to risks.

## Objective
To build a system that summarizes Terms & Conditions into simple,
user-friendly language using LLM-based processing.

## Features
- Text and URL-based input
- Simple, simplified, and risk-focused summaries
- API-based backend
- Interactive frontend
- Modular and scalable design

## Technology Stack
- Python
- FastAPI
- Streamlit
- LLM API (Mock/OpenAI/HuggingFace)
- Git & GitHub

## Conclusion
The system helps users quickly understand legal documents and highlights
important risks, improving transparency and user awareness.
